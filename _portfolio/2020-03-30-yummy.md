맛있는 여행의 아름다움, YUMMY
## 프로젝트 소개  
### 팀 구성  
 4인 팀 프로젝트
### 사용기술 및 언어    
  python 3.6.x  
  Django 3.0.x  
  MYSQL 8.0.x  
  Vue.js + vuetify  
### 개발 기간  
2020-03-30 ~ 2020-05-01(약 5주)


## 프로젝트 내용
### 주요 기능
 - 지도를 이용한 여행지역 선택 후, 음식점, 관광지, 숙박업소 정보 제공을 통한 일정 세우기 서비스 제공  
 - 지역별로 다른 사용자들의 여행계획 조회  
 - 다른 유저의 여행계획 스크랩 기능, CRUD  
 - 카카오톡 공유하기로 여행일정 공유  
 - tfidf를 이용한 유사 관광지 추천  
 - mf를 이용한 유저베이스 음식점 추천
